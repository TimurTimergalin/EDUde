Api приложения EdudE

На главную

С помощью библиотеки flask_restful было реализовано получение всей необходимой информации: список студентов у класса, классы учителя и ученика, список заданий для каждого класса, а так же ресурсы каждого пользователя, такие как имя и фамилия

Каждый запрос возвращается либо в виде списка всей необходимой информации, либо в виде привычного json формата, зависит от ситуации. В самом запросе содержатся объекты различных типов данных: просто текст, массив данных или классы
